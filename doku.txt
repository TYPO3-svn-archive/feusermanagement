Typoscriptoptionen:
==================
Abschnitt config:
-----------------
-adminMail / Addresse für Admin-Bestätigungslink
-adminMailSubject / Subject
-userMailSubject / Subject
-mailFromEMail / From Email
-mailFromName / From Name
-useMD5 / makes fe_users.password stored in md5
-requiredMarker / Template for the Required marker (little red star in the demo)
-usersFreshPid / Pid Where users are stored after creation
-usersFreshGroup / Group of fresh users
-usersConfirmedPid / pid of users with email verificated
-usersConfirmedGroup / group of users with email verificated
-usersAdminConfirmedPid / pid of users with admin confirmation
-usersAdminConfirmedGroup / group of users with admin confirmation
-userConfirmation / toggles email verification
-adminConfirmation / toggles admin verification
-autologin=1 / If no verification is set, users may be automatically logged in after registration
-autologinRedirPid=1 / Redirect Pid for autologin
-formNoSubmitAction=alert(alertMessage); / actionhandler for js-error if submit is neglected
-template / Template File

Optionen für jedes Feld:
----------------
-required / Input of a value is required
-type / Input type: possible Values: text/textarea/password/dropdown/radio/checkbox/hidden
-label / Label for the field. Either direkt input or prefix LL_field_<fieldname> takes the label from the locallang.db
-tooltip / Tooltip for the field. Either direkt input or prefix LL_field_<fieldname> takes the label from the locallang.db
-unique / Requires the value of the field to be unique in DB
-validation / commaseperated list of values (password/email/regexp) checks by regular expressions (password/email given, regexp takes the pattern from TS)
-jsvalidation / flag for inserting js to make the basic checks before sending the form to the server
-options / List of values/labels for dropdownfields and radioboxes
-value / preset value of the field
-notCheckedMessage / If the field is required this Message will be outputted if the field is not set
-onBlurValidation / Validates Field on Blur
